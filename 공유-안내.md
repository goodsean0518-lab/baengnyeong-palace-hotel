# 백령도 팰리스 호텔 리뉴얼 시안 — 공유(배포) 안내

데모(`index.html`)를 **공개 URL**로 만들어 카카오톡으로 링크 공유하는 방법.
단독 HTML이라 서버·빌드 불필요. 드래그 한 번이면 끝.

---

## 가장 쉬운 방법 — Netlify Drop (계정 없이 1분)

1. 브라우저에서 **https://app.netlify.com/drop** 열기
2. 이 폴더의 **`index.html` 파일을 화면 위로 드래그&드롭**
   - (이미지·폰트는 외부 CDN/위키미디어에서 자동 로딩 → 같이 안 올려도 됨)
3. 잠시 후 **공개 주소** 생성됨 → 예: `https://random-name-1234.netlify.app`
4. 그 주소가 바로 **카톡 공유 링크**. 복사해서 보내면 됨.

> 주소를 계속 유지하려면 Netlify 무료 가입(구글 로그인). 가입 안 하면 임시로 유지됨.
> 사이트 이름(주소 앞부분)도 무료로 변경 가능: 예) `baengnyeong-palace.netlify.app`

### 대안 (계정 있으면)
- **Vercel** (vercel.com) — 깔끔, 깃/계정 필요
- **Cloudflare Pages** — 빠름, 계정 필요
- **GitHub Pages** — 깃 사용 시

셋 다 무료. 비개발자면 **Netlify Drop**이 제일 쉬움.

## 카카오톡 공유 시 미리보기(썸네일)

OG 태그를 넣어둬서 카톡에 링크 붙이면 **두무진 사진 + 제목 카드**가 뜸.

- 미리보기가 안 뜨거나 옛날 걸로 뜨면 → 카카오가 캐시함.
- 새로고침: **카카오 공유 디버거** https://developers.kakao.com/tool/debugger/sharing
  → 주소 입력 → "초기화" (카카오 로그인 필요)

```
  실제 데이터로 교체 가능합니다.
```

## 공유 전 체크 (현재 상태)

| 항목 | 상태 |
|---|---|
| 호텔명·주소·전화 | ✅ 실제값 |
| 배편 정보 | ✅ 26.6 기준 실제값 (기상 변동 단서 포함) |
| 갤러리·히어로 사진 | ✅ 실제 백령도(위키/KOCIS, 데모용) |
| 객실·가격 | ⚠️ 예시 (워터마크 "시안" 표기됨) |
| 네이버 예약 링크 | ⚠️ 임시(booking.naver.com) — 실제 URL 교체 필요 |
| 지도 | ⚠️ 카카오 JS 키 넣어야 표시(현재 '카카오맵에서 보기' 링크는 작동) |

> 데모 좌하단 "시안(DEMO)" 배지가 있어 동료들이 시안임을 바로 인지함.
